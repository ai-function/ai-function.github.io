# svm

@giscus
giscus
2021 年 5 月 15 日
App
giscus
利用 GitHub Discussions 实现的评论系统，让访客借助 GitHub 在你的网站上留下评论和反应吧！本项目深受 utterances 的启发。

开源。🌏
无跟踪，无广告，永久免费。📡 🚫
无需数据库。所有数据均储存在 GitHub Discussions 中。:octocat:
支持自定义主题！🌗
支持多种语言。🌐
高可配置性。🔧
自动从 GitHub 拉取新评论与编辑。🔃
可自建服务！🤳
注意：
giscus 仍处于活跃开发中，GitHub 也还在活跃地开发 Discussions 及其 API，因此一些 giscus 的特性可能随时间损坏或变更。
